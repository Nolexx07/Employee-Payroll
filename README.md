# 💼 Employee Payroll Management System

## 📌 Overview
The **Employee Payroll Management System** is a Java-based application designed to calculate salaries for different types of employees using core **Object-Oriented Programming (OOP)** concepts such as **Inheritance, Polymorphism, and Abstraction**.

This project demonstrates how real-world payroll systems can be structured in a scalable and maintainable way.

---

## 🚀 Features
- Supports multiple employee types:
  - Full-Time Employees
  - Part-Time Employees
  - Interns
- Dynamic salary calculation based on employee type
- Clean and modular class design
- Demonstrates real-world OOP usage

---

## 🧠 OOP Concepts Used

### 🔹 Inheritance
- A base class `Employee` is created
- Child classes extend it:
  - `FullTimeEmployee`
  - `PartTimeEmployee`
  - `Intern`

### 🔹 Polymorphism
- Method `calculateSalary()` is overridden in each class
- Enables different salary calculations for different employee types

### 🔹 Abstraction
- `Employee` class is declared as abstract
- Defines common structure without implementation details
